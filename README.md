# bitter-Chocolate
ABOUT
-----
A premium dark-editorial e-commerce website built entirely with
pure HTML, CSS, and vanilla JavaScript.
No frameworks. No libraries. No dependencies.

PAGES
-----
index.html      Homepage — Hero, About, Products carousel, FAQ, Contact
shop.html       Shop — Full product list with filters and modal
cart.html       Cart — Items, quantity controls, order summary

FEATURES
--------
- Custom animated cursor with lag effect (desktop only)
- Full-screen preloader with sweep animation
- Sticky header with blur on scroll
- Dropdown navigation with mobile hamburger overlay
- Scroll reveal animations via IntersectionObserver
- Homepage product carousel with drag and swipe support
- FAQ accordion
- Contact form with reason chips
- Shop product list with category filters
- Product modal with quantity pills (minis) and stepper (signature)
- Cart persistence via localStorage
- Fully responsive — desktop, tablet, mobile
